# 产品B电商自动化系统

## 项目概述
产品B是本地部署的电商AI自动运营系统，面向商家私有数据喂养、学习中心、本地知识库和一句话任务拆解。

## 版本信息
- **当前版本：** v0.1 学习中心骨架版
- **发布日期：** 2026-07-02
- **依赖状态：** 已移除产品A硬依赖，可独立运行

## 技术栈
- **Python 3.8+**
- **FastAPI** - Web框架
- **Pydantic** - 数据验证
- **本地存储** - JSON文件存储（data/目录）

## 项目结构
```
product-b-ecommerce/
├── src/              # 源代码
│   ├── api/         # API接口
│   ├── models/      # 数据模型
│   ├── services/    # 业务服务
│   └── main.py      # 主程序
├── docs/            # 文档
├── config/          # 配置文件
├── data/            # 本地知识库（首次运行时自动生成）
└── README.md        # 本文件
```

## 核心功能（v0.1）
1. **学习中心** - 链接学习、文本学习、表格学习、复盘学习
2. **本地知识库** - JSON文件存储，支持知识沉淀和查询
3. **一句话任务拆解** - 自动识别任务意图并拆解为执行步骤
4. **无Key启动** - 支持无OPENAI_API_KEY基础启动

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行API服务器
```bash
python3 -m src.api.server
```

### 验证启动
```bash
# 健康检查
curl http://localhost:8000/health

# 学习中心健康检查
curl http://localhost:8000/api/learning/health

# API文档
curl http://localhost:8000/docs
```

## 安全说明
**高风险动作控制：**
- ✅ 只生成建议和待确认任务
- ❌ 不自动登录商家后台
- ❌ 不自动上架商品
- ❌ 不自动改价
- ❌ 不自动开车投放
- ❌ 不自动改预算
- ❌ 不自动扣费
